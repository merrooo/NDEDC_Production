“Adjusting your PATH environment”

👉 Select THIS option (VERY IMPORTANT):

✅ Git from the command line and also from 3rd-party software

Keep default options

Finish install

🔹 Step 3: RESTART EVERYTHING

This is NOT optional.

🔁 Restart:

VS Code

PowerShell / CMD

(Safest) Restart your PC

🔹 Step 4: Verify Git is installed

-------------------------------------------------------------------------------------------
Open VS Code → Terminal and run:

git --version

✅ You should see something like:

git version 2.44.0.windows.1

1-creat a new Repo in the GithuB without read me file
2-then put all these instructions in the git bash in VSC for example "Calculations" or the repo name

$ echo "# Calculations" >> README.md
$ git init
$ git add README.md
$ git commit -m "first commit"
$ git branch -M main
for example this branch repo https://github.com/merrooo/Calibrations.git
$ git remote add origin https://github.com/merrooo/Calibrations.git
$ git push -u origin main

note**
if he said 
$ git push -u origin main
remote: Repository not found.
fatal: repository 'https://github.com/merrooo/NDEDC_Smart_Meter.git/' not found
try to remove the repo like this
$ git remote remove origin
$ git remote add origin https://github.com/merrooo/NDEDC_Production.git
$ git remote -v

يجب أن يظهر:

origin  https://github.com/merrooo/NDEDC_Production.git (fetch)
origin  https://github.com/merrooo/NDEDC_Production.git (push)

$ git push -u origin main
💡 إذا ظهر خطأ آخر مثل unrelated histories مرة أخرى استخدم:

$ git pull origin main --allow-unrelated-histories
$ git push -u origin main
